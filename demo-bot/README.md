# MAXBoты Demo Bot #1 — FAQ / Support

Демонстрационный FAQ-бот для мессенджера MAX.

## Что умеет

- 10 тем FAQ с полными ответами (тарифы, сроки, примеры, интеграции, AI, и др.)
- Главное меню с быстрыми кнопками (inline keyboard)
- Распознавание ключевых слов в свободном тексте
- Long polling (без вебхука, запускается локально)

## Темы FAQ

| Тема | Ключевые слова |
|------|---------------|
| Цены и тарифы | цена, стоимость, тариф, прайс |
| Сроки | срок, когда, как долго |
| Что такое MAX | max мессенджер, что за мессенджер |
| Зачем бот | зачем бот, что умеет, польза |
| Примеры ботов | примеры, кейсы, портфолио |
| Консультация | консультация, связаться, менеджер |
| Требования | юрлицо, ООО, ИП, верификация |
| Поддержка | поддержка, sla, техподдержка |
| Интеграции | crm, 1с, битрикс, amocrm, api |
| AI-боты | ai, gpt, yandexgpt, нейросеть |

## Установка и запуск

```bash
cd demo-bot

# Установить зависимости
pip3 install -r requirements.txt

# Установить токен бота (получить у @MasterBot в MAX)
export MAX_BOT_TOKEN=your_bot_token_here

# Запустить
python3 bot.py
```

## Как получить токен

1. Откройте MAX messenger
2. Найдите бота `@MasterBot`
3. Следуйте инструкциям для создания нового бота
4. Скопируйте токен и установите его в `MAX_BOT_TOKEN`

## Расширение

Чтобы добавить новые FAQ-темы, отредактируйте `faqs.py`:

```python
{
    "id": "my_topic",
    "triggers": ["ключевое слово", "другой вариант"],
    "answer": "Текст ответа (поддерживает *markdown*)",
    "buttons": ["Кнопка 1", "Кнопка 2", "Главное меню"],
},
```
